## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing. 

### Prerequisites

What things you need to install the project software 

```
Linux, Mac OS Terminal or Windows Command Line (cmd); Git version control system.
```

### Cloning

for Linux 
```
$ cd /home/user/my_project
```

for macOS 
```
$ cd /Users/user/my_project
```

for Windows
```
$ cd C:/Users/user/my_project
```


and type:
```
$ git init
```
This creates a new subdirectory named .git that contains all of your necessary repository filesâ€‰.

Now clone the repository using git clone:
